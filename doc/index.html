Welcome to PowPy’s documentation!¶

Welcome to PowPy, the revolution of PowerShell but... In Python!

This is mainly a small framework to work with Windows Management Instrumentation API in local as well as in remote. The library is free to use, and it is firstly provided with a python interpreter to script in freestyle and monitor your Windows machines in remote.

Read the following documentation to learn more about PowPy.

Have fun!

Motivations¶

There wasn’t any motivations in the beginning. Indeed, this project had to be a 4 lines script Windows Management exercise in class, and turned to a WMI scripting framework by myself at home (That I didn’t want to through in the garbage). Since we have python, COM and WMI, the purpose of this framework is to give an interactive interface for system administrators to manage a local machine as a remote machine, or a set of machines in a specific domain, and so on.

The comparison between PowerShell and PowPy as to be taken with humour. This framework is based on PowerShell concepts, using the WMI API, to script and manage local or remote Windows machines on the fly, and only provide few fonctionnalities compared to Powershell (yet! =D).
